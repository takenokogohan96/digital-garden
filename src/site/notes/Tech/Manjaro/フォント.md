## 日本語の文字化けを解消
```
sudo pacman -S otf-ipafont
```
投入後再起動する

## Emojiの豆腐化を修正
```
sudo pacman -S noto-fonts-emoji unicode-emoji nodejs-emojione
```
投入後再起動する

## 韓国語の文字化けを解消
```
sudo pacman -S noto-fonts-cjk
```
投入後再起動する
